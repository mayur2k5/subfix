SubFix

This application lets you adjust timelag in a text-based subtitle file (SRT) and to merge two subtitle files into one
Adjust timelag
Please specify timelag value you want to add/substract

e.g. if you want to add 2 seconds type in 20
Merge files
Please specify the file and the corresponding timelag value
